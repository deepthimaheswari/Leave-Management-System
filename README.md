# Leave Management System

## Description

A role-based leave management system developed using Python Flask and SQLite for managing Student leave requests and approvals.

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## Features

- Student Login
- Apply for Leave
- View Leave Status
- Admin Approval/Rejection
- Student Management
- Email Notifications

## Project Structure

```text
Leave-Management-System
├── app.py
├── database.py
├── email_utils.py
├── leavedatabase.db
└── README.md
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/deepthimaheswari/Leave-Management-System.git
```

2. Move to project directory

```bash
cd Leave-Management-System
```

3. Install Flask

```bash
pip install flask
```

4. Run application

```bash
python app.py
```
